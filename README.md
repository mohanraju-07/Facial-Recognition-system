**Facial Recognition System**
===========================

**Project Overview**
-------------------

This facial recognition system uses a combination of the YOLO model and FaceNet to recognize faces in images and videos. The system is capable of generating a database of face embeddings and recognizing faces in real-time using a webcam.

**Folder Structure & Explanation**
---------------------------------

The project is organized into a single folder containing two Python scripts:

* `embeddings.py`: Generates a database of face embeddings using the YOLO model and FaceNet.
* `recognizewithcam.py`: Uses the database generated by `embeddings.py` to recognize faces in real-time using a webcam.

**Features**
------------

* Face detection and recognition using the YOLO model and FaceNet
* Generation of a database of face embeddings
* Real-time face recognition using a webcam
* Python-based implementation for ease of use and modification

**Technologies Used**
---------------------

* OpenCV for computer vision and image processing
* FaceNet for face recognition
* YOLO (You Only Look Once) for object detection
* Python 3.x for scripting

**How to Run the Project**
---------------------------

### Prerequisites

* Install the required libraries by running `pip install opencv-python facenet-pytorch`
* Ensure that a webcam is properly connected to the system

### Running the Script

1. Navigate to the project folder in the terminal/command prompt.
2. Run `python embeddings.py` to generate the database of face embeddings.
3. Run `python recognizewithcam.py` to recognize faces in real-time using the webcam.
